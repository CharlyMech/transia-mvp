# Guía de Uso - Transia MVP

Bienvenido a la guía de uso de la aplicación Transia.

## Gestión de Conductores

### Ver Perfil
Desde la pestaña "Conductores", pulse sobre una tarjeta de conductor para ver su detalle.

### Registro de Horas
1.  Navegue al detalle del conductor.
2.  Pulse en "Registro horario".
3.  Seleccione un día en el calendario.
4.  Use los botones de "Iniciar jornada", "Pausar" o "Finalizar" para control en tiempo real.
5.  Use los botones "+", Editar o Eliminar para modificar rangos de tiempo manualmente.

> [!NOTE]
> El sistema cierra automáticamente los registros abiertos del día anterior a las 23:59.

## Reportes e Incidencias

### Crear un Reporte
1.  Vaya a la pestaña "Reportes".
2.  Pulse el botón "Nuevo (+)".
3.  Seleccione el tipo de reporte.
4.  Añada una nota descriptiva (mínimo 10 caracteres).
5.  Adjunte fotos de la incidencia (cámara o galería).
6.  Marque "Incluir ubicación" para geolocalizar el reporte.
7.  Pulse "Crear reporte".

## Filtros y Búsqueda
En las listas (Conductores, Flota), use la barra de búsqueda superior para encontrar elementos por nombre, teléfono o matrícula.
Despliegue el panel de "Filtros y ordenación" para filtrar por Estado o cambiar el orden de la lista.
