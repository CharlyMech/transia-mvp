# Usage Guide - Transia MVP

Welcome to the Transia application usage guide.

## Driver Management

### View Profile
From the "Drivers" tab, tap on a driver card to view their details.

### Time Registration
1.  Navigate to the driver's detail.
2.  Tap on "Time Registration".
3.  Select a day on the calendar.
4.  Use the "Start Shift", "Pause", or "End Shift" buttons for real-time control.
5.  Use the "+", Edit, or Delete buttons to modify time ranges manually.

> [!NOTE]
> The system automatically closes open registers from the previous day at 23:59.

## Reports & Incidents

### Create a Report
1.  Go to the "Reports" tab.
2.  Tap the "New (+)" button.
3.  Select the report type.
4.  Add a descriptive note (minimum 10 characters).
5.  Attach photos of the incident (camera or gallery).
6.  Check "Include location" to geolocate the report.
7.  Tap "Create Report".

## Filters & Search
In lists (Drivers, Fleet), use the top search bar to find items by name, phone, or license plate.
Expand the "Filters & Sort" panel to filter by Status or change the list order.
